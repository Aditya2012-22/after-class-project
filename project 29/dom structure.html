Accessing Values with DOM

The DOM (Document Object Model) allows JavaScript to interact with HTML elements. Below are the main ways to access values using the DOM.

1. Accessing an Element by ID

const title = document.getElementById("title").innerText;

2. Accessing Elements by Class

const items = document.getElementsByClassName("item");
console.log(items[0].innerText);

3. Using Query Selector

const button = document.querySelector("#btn");
const firstItem = document.querySelector(".item");

4. Accessing Input Values

const inputValue = document.getElementById("userInput").value;

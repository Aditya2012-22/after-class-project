CSS Properties and Values

Example Box

In CSS, properties define what you want to style, and values define how you want to style it.

Examples: • color: red; → changes text color to red.
• font-size: 20px; → makes text larger.
• background-color: yellow; → adds background color.
• margin: 10px; → adds space outside an element.
• padding: 15px; → adds space inside an element.

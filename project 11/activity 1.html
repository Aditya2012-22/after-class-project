CSS Cascading Effect

This paragraph uses the default paragraph style (blue color).

This paragraph uses a class selector (green text, bold).

This paragraph uses an ID selector (orange text, overrides class).

This paragraph uses an inline style (red text, highest priority).

Notice how CSS applies rules in order: 1. Inline styles > 2. ID selectors > 3. Class selectors > 4. Element selectors. This is why the last paragraph appears red even though other rules exist.
